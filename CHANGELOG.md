# Changelog

All notable changes to `expready` are documented in this file.

The format follows Keep a Changelog, and this project uses SemVer.

## [0.1.1] - 2026-04-13

### Added
- Dedicated unit tests for preflight helpers and grouped error formatting.

### Changed
- Improved preflight input error output with grouped sections and actionable hints.

## [0.1.0] - 2026-04-11

### Added
- Initial public release.
- Core CLI commands: `validate` and `fix`.
- Metadata, design, and cross-file validation with HTML reporting.
- Input preflight checks for common file and schema issues.
- Manifest-aware sample/path consistency checks.
- Test suite, CI workflow, and release/build support files.
